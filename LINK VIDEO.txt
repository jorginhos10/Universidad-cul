SISTEMA CRUD DE TAREAS

Descripción del Proyecto
Este proyecto es un sistema CRUD (Crear, Leer, Actualizar, Eliminar) para la gestión de tareas. Permite a los usuarios registrar nuevas cuentas, iniciar sesión, y gestionar tareas personales mediante una interfaz intuitiva. Los usuarios pueden crear, leer, actualizar y eliminar tareas, proporcionando una solución completa para la organización y seguimiento de sus responsabilidades diarias. La aplicación está construida utilizando Angular para el frontend y NestJS para el backend, garantizando una experiencia de usuario fluida y una gestión eficiente de los datos.
